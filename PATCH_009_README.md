# Patch 009 — Domain Architecture Part 1

## Purpose

Patch 009 begins the formal AgriOS domain architecture. It translates the approved BRD baseline into domain boundaries, bounded contexts, ownership rules, context relationships, aggregates, domain services, domain events, shared-kernel constraints, and initial context-map decisions.

## Included documents

- Domain architecture overview
- Domain classification and subdomains
- Bounded-context catalogue
- Context map
- Core, supporting, and generic subdomains
- Aggregate and entity starter catalogue
- Domain-event catalogue
- Domain-service and policy catalogue
- Ownership and data-authority matrix
- Shared-kernel and integration-boundary rules
- Anti-corruption-layer guidance
- Initial module map for a Spring Modulith implementation
- Open domain decisions
- Traceability from BRD capabilities to bounded contexts

## Apply

Extract this ZIP into the AgriOS repository root. All archive paths are repository-relative and there is no wrapper directory.
