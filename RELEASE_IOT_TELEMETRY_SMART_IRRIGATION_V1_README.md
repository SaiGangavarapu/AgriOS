# AgriOS IoT Device, Telemetry, and Smart Irrigation Automation Core v1.0

Implemented:

- IoT device registry, provisioning, activation, assignment, heartbeat, and health
- field and water-source device assignment
- telemetry stream registry
- normalized telemetry ingestion
- range validation and telemetry alerts
- latest-reading APIs
- irrigation automation policies
- advisory, approval-required, semi-automatic, and full-automatic control modes
- soil-moisture and rain threshold decisions
- explicit decision records and confidence
- approval workflow
- idempotent actuator commands
- emergency-stop-ready safety model
- Flyway migrations V019 through V021
