# AgriOS Crop Cycle, Seed, Farm Operations, and Task Calendar Core v1.0

This cumulative implementation adds:

- crop-cycle creation from approved crop plans
- crop-cycle activation, sowing, stage tracking, crop-loss records, and season closure
- seed-lot inventory, germination testing, treatment, and crop-cycle allocation
- farm-operation planning, inputs, start, completion, and crop-cycle history
- domain-owned tasks with a consolidated task/calendar API
- Flyway migrations V011 through V014
- module boundary declarations and integration-test coverage

The ZIP is cumulative and preserves the previous repository baseline.
