# Patch 012 — SRS Part 1

## Purpose

Patch 012 begins the AgriOS System Requirements Specification. It establishes SRS document control, actor definitions, use-case conventions, functional and non-functional requirement formats, acceptance-criteria rules, and detailed requirements for farmer onboarding, farm registration, field management, tenure, roles, delegated access, and consent.

## Included documents

- SRS document control
- Actor catalogue
- Use-case catalogue
- Functional requirement conventions
- Non-functional requirement conventions
- Acceptance-criteria conventions
- Farmer onboarding requirements
- Farmer profile and verification requirements
- Household and organization requirements
- Farm registration requirements
- Field and boundary requirements
- Tenure and cultivation-right requirements
- Role and delegated-access requirements
- Consent and data-sharing requirements
- Validation and error catalogue
- Acceptance criteria
- Traceability matrix

## Apply

Extract this ZIP into the AgriOS repository root. All paths are repository-relative and there is no wrapper directory.
