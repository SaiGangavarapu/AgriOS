# AgriOS Physical Database Foundation v1.0

This cumulative repository release preserves the existing AgriOS documentation
baseline and adds the initial executable PostgreSQL/PostGIS database foundation.

Included:

- PostgreSQL and PostGIS bootstrap
- module-owned schemas
- baseline tenant, identity, farmer, farm, field, consent, audit, and outbox tables
- core indexes and constraints
- Flyway ordering guidance
- verification SQL
- Docker Compose database service
- environment template
- physical database implementation guide

The local `.git` directory is not included in this archive.
