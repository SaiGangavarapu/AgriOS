# BRD 35 — Input Application and Safety Controls

## 1. Objective

Prevent unsafe, duplicate, mistimed, incompatible, or unsupported applications of fertilizers and soil amendments.

## 2. Supported input categories

- mineral fertilizer
- manure
- compost
- vermicompost
- green manure
- biofertilizer
- microbial inoculant
- lime
- gypsum
- sulphur amendment
- micronutrient
- foliar nutrient
- fertigation product
- other approved soil or crop input

## 3. Input catalogue attributes

- product name
- manufacturer
- nutrient analysis
- formulation
- batch or lot
- registration or approval status
- permitted farming systems
- storage requirements
- incompatibilities
- application methods
- safety instructions
- expiry
- evidence source

## 4. Pre-application validations

- correct field and crop
- correct crop stage
- approved nutrient plan
- valid quantity and unit
- recent duplicate application
- minimum interval
- weather suitability
- soil-moisture condition
- irrigation availability
- product expiry
- certification compatibility
- operator authorization

## 5. Micronutrient safeguards

Micronutrient application should require:

- current deficiency evidence
- crop relevance
- source and dose validation
- interaction check
- accumulation warning
- repeat-application history

## 6. Application record

- field and zone
- crop cycle
- product and lot
- quantity
- nutrient equivalent
- application date and time
- method
- operator
- equipment
- weather
- irrigation
- cost
- evidence
- observed result

## 7. Business requirements

- BR-143: The platform shall maintain a governed input catalogue.
- BR-144: The platform shall validate input applications before completion.
- BR-145: Micronutrient use shall require evidence and repeat-use checks.
- BR-146: Application records shall include lot, quantity, method, operator, and cost.
- BR-147: The platform shall identify duplicate and incompatible applications.
