# BRD 08 — Success Measures and KPIs

## 1. Adoption

- registered farmers
- active farmers
- registered fields
- active crop cycles
- weekly active users
- advisory acknowledgement rate

## 2. Operational use

- planned tasks completed on time
- farm operations recorded
- soil tests linked to fields
- crop plans approved
- field visits completed
- expert-review turnaround time

## 3. Advisory quality

- percentage with complete evidence
- percentage with confidence
- low-confidence escalation rate
- expert override rate
- farmer-reported usefulness
- advisory correction rate

## 4. Agronomic outcomes

Pilot evaluation may measure:

- reduction in avoidable input applications
- improvement in irrigation timing
- change in water use
- change in nutrient-use efficiency
- yield stability
- reduction in preventable weather loss

These metrics require controlled interpretation and must not attribute all outcome changes solely to AgriOS.

## 5. Economic outcomes

- input cost per field
- gross margin
- loss avoided
- price realization
- transaction cost reduction
- expert-service efficiency

## 6. Inclusion

- local-language usage
- voice usage
- women operators
- tenant-farmer participation
- offline transaction success
- low-bandwidth completion rate

## 7. Platform quality

- synchronization success
- alert delivery latency
- API availability
- device data completeness
- sensor health
- support resolution time
- security incidents
- consent violations

## 8. Pilot exit criteria

The pilot should not scale until:

- farmer usability is validated
- expert workflow is sustainable
- critical advisories are traceable
- offline reliability is acceptable
- device quality is understood
- governance review is complete
