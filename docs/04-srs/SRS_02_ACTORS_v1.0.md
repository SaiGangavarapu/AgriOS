# SRS 02 — Actors

Primary actors:

- Farmer
- Household Operator
- Farm Worker
- Field Officer
- Agronomist
- Soil Scientist
- Plant Pathologist
- Entomologist
- Laboratory Operator
- FPO Manager
- Buyer
- Insurer
- Lender
- Administrator
- Support Operator
- External Weather Provider
- External IoT Gateway

Each actor shall operate only within role, tenant, programme, farm, field, consent, and time scope.
