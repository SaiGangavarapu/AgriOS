# SRS 08 — Household and Organization

## FR-HOUSEHOLD-001 Create household

The system shall allow an authorized user to create a household and designate a primary reference farmer.

## FR-HOUSEHOLD-002 Add member

The system shall allow household members to be added with role and validity period.

## FR-HOUSEHOLD-003 End membership

The system shall end membership without deleting history.

## FR-ORG-001 Register organization

The system shall register FPOs, cooperatives, SHGs, NGOs, and other supported organizations.

## FR-ORG-002 Add membership

The system shall add a farmer or user to an organization with role, status, start date, and verification source.

## FR-ORG-003 Multiple memberships

The system shall support multiple active organization memberships.

## FR-ORG-004 Membership access boundary

The system shall not infer data access solely from organization membership.

## VR-HOUSEHOLD-001 Duplicate active membership

The system shall prevent duplicate active membership for the same relationship unless explicitly allowed.

## DR-HOUSEHOLD-001 Membership history

The system shall preserve start, end, role, and verification history.
