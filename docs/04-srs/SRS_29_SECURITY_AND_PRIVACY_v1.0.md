# SRS 29 — Security and Privacy

- SR-001: Support configurable authentication and account recovery.
- SR-002: Enforce least privilege and contextual authorization.
- SR-003: Require reauthentication for sensitive actions.
- SR-004: Support session listing and revocation.
- SR-005: Separate farmer identity from user account.
- SR-006: Support purpose-specific, revocable consent.
- SR-007: Minimize shared data.
- SR-008: Protect offline data and sensitive documents.
- SR-009: Audit privileged actions, exports, sharing, and automation.
- SR-010: Support incident detection, containment, investigation, notification, and corrective action.
- SR-011: Support retention, access, correction, export, and deletion workflows.
