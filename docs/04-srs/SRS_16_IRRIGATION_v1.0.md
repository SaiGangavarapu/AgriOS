# SRS 16 — Irrigation

- FR-IRR-001: Generate stage-aware schedules using soil, weather, crop, source, and method.
- FR-IRR-002: Output irrigate, delay, or skip; depth; volume; runtime; reason; confidence; and reassessment.
- FR-IRR-003: Support surface, furrow, basin, sprinkler, drip, and micro-irrigation.
- FR-IRR-004: Record actual start, end, source, volume, flow, runtime, cost, and interruptions.
- FR-IRR-005: Distinguish measured and estimated volume.
- FR-IRR-006: Maintain rainfall, irrigation, use, runoff, and drainage accounting.
- FR-IRR-007: Enforce automation safeguards.
