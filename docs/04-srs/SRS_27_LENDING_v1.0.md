# SRS 27 — Lending

- FR-LEND-001: Create purpose-specific lender evidence requests.
- FR-LEND-002: Let the farmer review scope before sharing.
- FR-LEND-003: Include source, date, verification, completeness, and self-declared status.
- FR-LEND-004: Require active consent.
- FR-LEND-005: Make access time-bound and revocable.
- FR-LEND-006: Never autonomously approve or reject credit.
- FR-LEND-007: Never infer legal ownership from cultivation records.
