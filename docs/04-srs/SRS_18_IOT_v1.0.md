# SRS 18 — IoT

- FR-IOT-001: Register devices, sensors, actuators, gateways, firmware, owner, operator, and lifecycle.
- FR-IOT-002: Record installation, depth, orientation, location, calibration, and commissioning.
- FR-IOT-003: Preserve assignment history.
- FR-IOT-004: Ingest telemetry with observed and received time, value, unit, battery, signal, and quality.
- FR-IOT-005: Detect range errors, stuck values, gaps, duplicates, drift, and clock issues.
- FR-IOT-006: Support offline buffering and replay.
- FR-IOT-007: Track maintenance, replacement, calibration, and health.
- FR-IOT-008: Enforce command expiry, idempotency, manual override, runtime limits, flow confirmation, dry-run protection, and rain lockout.
