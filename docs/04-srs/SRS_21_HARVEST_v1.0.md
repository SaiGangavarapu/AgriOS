# SRS 21 — Harvest

- FR-HARVEST-001: Determine readiness using crop stage, maturity, moisture, weather, labour, machinery, storage, and buyer constraints.
- FR-HARVEST-002: Create harvest plans with window, method, expected quantity, quality, and contingency.
- FR-HARVEST-003: Record date, area, quantity, moisture, grade, loss, labour, machinery, cost, and destination.
- FR-HARVEST-004: Support multiple and partial harvests.
- FR-HARVEST-005: Create produce lots from harvest events.
