# Patch 010 — Domain Architecture Part 2

## Purpose

Patch 010 expands the AgriOS domain architecture with detailed context specifications, commands, queries, process managers, repositories, ports, workflows, invariants, consistency boundaries, and failure-handling rules.

## Included documents

- Detailed context specification template
- Farmer, farm, field, and tenure context specifications
- Soil, water, and agronomy knowledge context specifications
- Crop planning, crop cycle, seed, operations, and task context specifications
- Nutrient, irrigation, weather, IoT, telemetry, and advisory context specifications
- Pest, harvest, traceability, market, economics, insurance, and lending context specifications
- Identity, consent, notification, localization, configuration, audit, and support context specifications
- Command catalogue
- Query catalogue
- Repository and port catalogue
- Process managers and sagas
- Consistency, idempotency, and failure-handling rules
- Domain architecture traceability update

## Apply

Extract this ZIP into the AgriOS repository root. All archive paths are repository-relative and there is no wrapper directory.
